mysql5.7安装

1、下载mysql

　　5.7 32位：https://dev.mysql.com/get/Downloads/MySQL-5.7/mysql-5.7.19-win32.zip


　　5.7 64位：https://dev.mysql.com/get/Downloads/MySQL-5.7/mysql-5.7.19-winx64.zip


2、解压到自己的目录

3、添加环境变量

　　把D:\Program Files\mysql-5.7.19\bin添加到path后面

4、新建my.ini文件

　　在D:\Program Files\mysql-5.7.19文件夹下面新建my.ini文件，编码为ansi，文件内容如下:

5、安装mysql

　　在cmd命令中(管理员权限，不要用Windows PowerShell，这个玩意不能查看报错日志)，进入D:\Program Files\mysql-5.7.19\bin目录下，输入mysqld –install即可安装；安装完成会显示 Service successfully installed。

6、初始化数据文件

　　输入mysqld –initialize-insecure –user=mysql;即可初始化

7、启动mysql服务

　　输入net start mysql即可启动mysql服务

8、登陆mysql

输入mysql –u root –p然后回车，显示输入密码，直接回车即可。

9、更改root密码

　　输入:update mysql.user set authentication_string=password(‘liguoqing’) where
user=’root’ and Host = ‘localhost’;

10、刷新权限

　　输入:flush privileges

11、修改my.ini文件

　　找到刚才新增的my.ini文件，删除skip-grant-tables这一行，注释掉也可以

12、重启mysql服务

输入:exit 先退出数据库

停止mysql服务:net stop mysql;

启动mysql服务:net start mysql;

（不要直接net restart mysql）
